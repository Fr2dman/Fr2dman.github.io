Smart AC Simulator & SAC Agent Generator

Role RL Engineer, PM
Period 2025.07.07 ~ 2025.08.18

Tech Python · RL · MQTT

Project Overview

멀티존(4존) 실내 환경을 모사하는 경량 HVAC 시뮬레이터와 이를 제어하기 위한 SAC(Soft Actor-Critic) 강화학습 에이전트 생성 파이프라인입니다. 에너지 효율과 재실자의 쾌적도를 동시에 만족시키는 최적의 제어 정책을 학습하는 것을 목표로 합니다.

Key Features

  • Multi-zone Physics Engine: 존 에너지/수분 수지, 공기 혼합, 열전달 등을 고려한 정밀 시뮬레이션
  • Gymnasium Environment: 강화학습 표준 인터페이스(Gymnasium) 호환 환경 제공
  • SAC Agent Training: Stable Baselines3를 활용한 SAC 알고리즘 학습 파이프라인
  • Reward Engineering: 에너지 소비 최소화 및 쾌적도(PMV 등) 최대화를 위한 복합 보상 함수 설계

My Role & Contributions

이 프로젝트에서 시뮬레이터 개발 및 강화학습 모델링을 담당했습니다.

  • HVAC 물리 엔진(열역학적 모델링) 구현 및 검증
  • Custom Gym Environment 설계 및 구현
  • SAC 하이퍼파라미터 튜닝 및 학습 성능 최적화
  • 다양한 시나리오 기반의 테스트 스위트 작성
