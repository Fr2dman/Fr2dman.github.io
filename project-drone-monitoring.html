Drone Monitoring System

Real-time drone status tracking and mission control dashboard

Role Full Stack Developer
Period 2025.03.21 ~ 2025.03.28
Team 1
Tech Node.js · Vue.js · WebSocket

Project Overview

다수의 드론 상태를 실시간으로 추적하고 미션을 할당할 수 있는 웹 기반 통합 관제 시스템입니다. TCP 소켓 통신을 통해 드론 시뮬레이터와 데이터를 주고받으며, WebSocket을 이용해 웹 클라이언트에 실시간으로 정보를 업데이트합니다.

Key Features

  • Real-time Tracking: 지도 상에서 드론의 실시간 위치 및 상태(배터리, 속도 등) 시각화
  • Mission Control: 드론에게 목표 지점 이동 등 미션 할당 및 제어
  • Drone Simulation: 가상의 드론 클라이언트를 통해 시스템 기능 검증 및 테스트
  • Responsive Dashboard: 다양한 디바이스에서 접근 가능한 반응형 웹 인터페이스

My Role & Contributions

이 프로젝트에서 풀스택 개발을 수행했습니다.

  • Node.js(Express) 기반의 백엔드 서버 구축 (TCP/WebSocket/REST API 통합)
  • Vue.js를 이용한 프론트엔드 대시보드 및 지도 시각화 구현
  • 드론 시뮬레이터 클라이언트 로직 개발
  • 실시간 데이터 처리 파이프라인 최적화
