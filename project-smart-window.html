Smart Window (Lumiscape)

IoT-based smart window system with opacity control

Role Qt/QML Developer
Period 2025.10.13 ~ 2025.11.20
Team 6
Tech Qt/QML · C++ · MQTT

Project Overview

Lumiscape는 투명도 조절이 가능한 스마트 윈도우 시스템입니다. 사용자는 외부 날씨나 개인 선호에 따라 창문의 투명도를 조절하고, 주변 조명과 연동하여 최적의 실내 분위기를 조성할 수 있습니다.

Key Features

  • Opacity Control: PDLC 필름을 이용한 창문 투명도 미세 조절
  • Ambient Lighting: 창문 주변 LED 조명 제어 및 무드등 기능
  • Local Touch Interface: 임베디드 터치스크린을 통한 직관적인 제어
  • IoT Connectivity: 모바일 앱 및 서버와의 연동

My Role & Contributions

이 프로젝트에서 Qt/QML을 활용한 로컬 제어 인터페이스 개발을 전담했습니다.

  • 임베디드 리눅스 환경에서의 Qt Quick (QML) 기반 UI/UX 설계 및 구현
  • C++ 백엔드 로직과 QML 프론트엔드 간의 데이터 바인딩 및 통신
  • 터치 이벤트 처리 및 부드러운 애니메이션 효과 구현
  • 하드웨어 제어 모듈(GPIO 등)과의 인터페이스 연동
